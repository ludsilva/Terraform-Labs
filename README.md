# Terraform-Labs

Este repositório tem como objetivo reunir meus laboratórios e projetos de estudo com o Terraform, com foco na prática de Infraestrutura como Código (IaC) utilizando AWS (e LocalStack para economia de custos) e boas práticas de ambientes cloud.

O objetivo é criar uma base sólida de conhecimento em Terraform, explorando diferentes serviços e recursos da AWS, além de organizar a minha progressão nos estudos, partindo de conceitos básicos até arquiteturas mais complexas.

### Laboratórios

Os laboratórios estão organizados em pastas independentes, cada uma contendo sua própria documentação e arquivos Terraform.

- [Lab 1 - EC2 com LocalStack](./lab1-ec2-localstack/)
- [Lab 2 - VPC + EC2 + Security Groups com LocalStack](./lab2-vpc-ec2-localstack/)
